# HebbCode

> 🚧 Under Construction

**HebbCode** is an AI agent project named after Hebb's law:

> *"Cells that fire together, wire together."*
> — Donald Hebb, *The Organization of Behavior* (1949)

Hebb's law describes the principle of synaptic plasticity — neurons that fire simultaneously strengthen their connections over time. We map this idea to the collaboration between code and AI agents: modules, tools, and agents that work together frequently form tighter, more efficient circuits.

The project is in its early stages. Stay tuned.
